1. Сколько типов данных существует в JS?

  • Числа
  • Строки
  • Символ
  • Логический
  • null
  • undefined
  • Объект

2. Как вывести информацию в консоль?

Использовать констуркцию: 
console.log(appData);

3. Какая функция операторов || и &&?

Сравнение данных и передача результата в логическом типе данных.

При || будет выдавать true если хотя бы одно сравниваемое выражение верно.

При && выдаст false если хотябы одно сравниваемое выражение не верно.
